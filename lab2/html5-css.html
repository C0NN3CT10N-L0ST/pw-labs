HTML5 e CSS

Esta página contém informações sobre HTML5 e CSS.

HTML5

HTML (HyperText Markup Language) é uma linguagem que permite definir a estrutura de uma página web.

Elemento Utilizado Descrição
header ✅ Utilizado no início de cada uma das páginas para definir o cabeçalho que contém o conteúdo introdutório da mesma.
nav ✅ Utilizado em cada uma das páginas para representar uma secção de links de navegação.
main ✅ Utilizado em todas as páginas para definir o conteúdo principal das mesmas.
aside ❌ -
footer ✅ Utilizado em todas as páginas para definir o autor.
section ✅ Utilizado em algumas páginas para representar secções de informação e que contêm um 'heading'.

CSS

CSS (Cascading Style Sheets) é uma linguagem que permite estilizar componentes HTML numa página web.

Tipo de Seletor Utilizado Descrição
universal ✅ Utilizado para alterar a fonte em todos os elementos.
tipo ✅ Utilizado para aplicar a cor de fundo em todas as páginas através do elemento 'body'.
classe ✅ Utilizado para centrar os ícones desta table.
id ✅ Utilizado para alterar o tamanho da fonte do texto da introdução.
atributo ✅ Utilizado para alterar a cor do atributo '_blank' das hiperligações.
pseudo-classe ✅ Utilizado para alterar o estado das âncoras.
pseudo-elemento ✅ Utilizado para alterar o tamaho da primeira letra do texto da introdução.

Combinador de Seletor Utilizado Descrição
descendente ✅ Utilizado para colocar todos as perguntas ('p' dentro de 'div') do quizz a negrito.
filho ✅ Utilizado para alterar o elemento 'legend' dentro do elemento 'fieldset'.
adjacente ❌ -
imediatamente adjacente ❌ -
agrupar ✅ Utilizado para aplicar bordas nos elementos 'table', 'th' e 'td'.
